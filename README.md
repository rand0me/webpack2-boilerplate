webpack2-boilerplate
====================
webpack 2, yarn, standard, standard-version, commitizen, cz-conventional-changelog minimalist boilerplate
----------------------------------------------------------------------------------------------

## Getting Started
```
git clone --depth=1 https://github.com/rand0me/webpack2-boilerplate my-project-name
cd my-project-name
rm -rf .git
git init
```

## Developing
```
yarn start
```
or
```
npm start
```

## Building
```
yarn build
```
or
```
npm run build
```
